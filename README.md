## How to Play

1. Download or Clone the source code.
2. Import the game in Godot. Build & Play.


## System Requirements

- OS: Windows 7/8/10 64 bit, macOS,
- CPU: Dual-core processor, 2.0 GHz
- RAM: 1GB
- Storage: 50 MB available space
